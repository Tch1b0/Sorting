# Sorting _in C_

Just a simple implementaion of sorting algorithms in [C](<https://en.wikipedia.org/wiki/C_(programming_language)>)

## Implemented algorithms

- [x] Bubble sort
- [x] Gnome sort
- [ ] Bogo sort
- [ ] Quick sort
- [ ] Merge sort

## Tasks

`task build`:
Build the project into the `./build` directory

`task run`:
Run the executable

`task br`:
Build & Run the program

## Requirements

- gcc
- taskfile
